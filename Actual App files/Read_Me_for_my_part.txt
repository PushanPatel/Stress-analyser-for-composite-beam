Run Matlab in your system (preferably R2020a version)


Open composite_beam_2.mlapp to open the app designer file which was created to make the application. 
Standalone file is based on this file with other files as additional requirements

open output.mlapp to open app designer file for output window. Note: during actual app execution this app 
window is opened when user presses calculate button. Run composite_beam_2.mlapp in app designer. Directly running this file will of no use

To install matlab app
open Stress analyser in composite beam with combined loading.mlappinstall file to install this app as a matlab package
